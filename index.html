git/github help

git

Ορισμός ταυτότητας

Για τον ορισμό ταυτότητας εκτελούμε την εντολή:

$ git config --global user.name "John Doe"
$ git config --global user.email johndoe@example.com

Για τον έλεγχο του user.name εκτελούμε την εντολή:

git config user.name

Για τον έλεγχο του user.email εκτελούμε την εντολή:

git config user.email

Αρχικοποίηση αποθετηρίου

Για την αρχικοποίηση αποθετηρίου σε υπάρχοντα κατάλογο εκτελούμε την εντολή:

git init

Έλεγχος της κατάστασης των αρχείων

Για τον έλεγχο της κατάστασης των αρχείων εκτελούμε:

git status

Για πιο σύντομη περιγραφή:

git status -s

Καταχώρηση τροποποιημένων αρχείων

Για την καταχώρηση αρχείων στο αποθετήριο εκτελούμε την εντολή:

git add home.html

Η καταχώρηση αφορά Modified ή/και Untracked αρχεία.

Για ταυτόχρονη καταχώρηση πολλών αρχείων (new, modified, και deleted):

git add --all

Υποβολή αλλαγών

Για την υποβολή αλλαγών εκτελούμε την εντολή:

git commit -m "message here"

Πρέπει να προστεθεί σχετικό μήνυμα με την εντολή commit.

Ιστορικό υποβολών

Για την εμφάνιση του ιστορικού υποβολών εκτελούμε την εντολή:

git log

Κλωνοποίηση αποθετηρίου

Για την κλωνοποίηση αποθετηρίου εκτελούμε την εντολή:

git clone https://github.com/username/reponame

Δημιουργείται ο φάκελος reponame

Δεν χρειάζεται αρχικοποίηση (δημιουργείται το σχετικό .git)

Για κλωνοποίηση αποθετηρίου σε νέο φάκελο: 

git clone https://github.com/username/reponame newfolder

Ενημέρωση τοπικού αποθετηρίου από απομακρυσμένο

Για την ενημέρωση τοπικού αποθετηρίου από απομακρυσμένο εκτελούμε την εντολή:

git pull
